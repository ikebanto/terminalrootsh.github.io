Se você acabou de instalar o Debian Jessie 8, certifique-se que você já atualizou os repositórios. Feito isso, abra o Terminal (Aplicações, Acessórios, Terminal), na GUI MATE. Adquira as permissões de root: {% highlight bash %} $ su {% endhighlight %} Instale o pacote de atualizações do Iceweasel para Português do Brasil. {% highlight bash %} # apt-get install iceweasel-l10n-pt-br {% endhighlight %} Pronto! Agora é só reiniciar o Iceweasel. Baseado em: http://blog.tiagopassos.com/
