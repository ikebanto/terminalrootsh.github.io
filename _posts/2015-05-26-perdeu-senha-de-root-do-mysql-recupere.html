Pare o MySQL:

$ sudo service mysql stop
Inicie o modo seguro:

$ sudo mysqld_safe –skip-grant-tables
Abra outro terminal e execute:

$ mysql -u root
Depois:

MYSQL>

UPDATE mysql.user SET Password=PASSWORD(‘novasenha’) WHERE User=’root';
Inicie o MySQL:

$ sudo service mysql stop
Fim.
