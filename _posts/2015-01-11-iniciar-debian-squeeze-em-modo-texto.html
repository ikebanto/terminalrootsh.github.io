Dica rápida!

Para iniciar em modo texto:
{% highlight bash %}# update-rc.d -f gdm3 remove{% endhighlight %}

Para iniciar o gráfico separadamente:
{% highlight bash %}$ startx{% endhighlight %}

Para voltar o padrão para modo gráfico
{% highlight bash %}# update-rc.d -f gdm3 defaults{% endhighlight %}

Somente!
