Fazer coisas básicas e aos poucos fazem memorização e fixação melhor dos detalhes.Estou postando 2 exemplos de loop simples em for, ambos no mesmo código, como compilar, está nos posts antigos sobre C, é só pesquisar, breve passarei teorias sobre tudo. PasteBin: CódigoNoBlog: {% highlight bash %} #include int main(){ int i; char letra; for(i=1;i<=10;i++) printf("%d\n",i); for(letra = 'A';letra<='Z';letra++) printf("%c\n",letra); return(0); } {% endhighlight %}
