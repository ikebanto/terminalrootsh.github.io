Agora uma nova forma de instalação de distribuições Linux está disponível para usuários iniciantes que usam Windows. Trata-se de um executável para Windows que é baixado da internet e auxilia o usuário no processo de instalação do Linux.

A justificativa do projeto é que muitos usuários ainda encontram dificuldades de "queimar" uma imagem de CD ou particionar um disco rígido, dai nasce um executável que elimina essas etapas tornando o processo de instalação de Linux bem mais simples.

Se você possui um Microsoft Windows e quer instalar o Linux sem muito trabalho, basta acessar esse site e clicar no Instalador.exe
que todo o processo de instalação do Linux será automático!

http://goodbye-microsoft.com/
