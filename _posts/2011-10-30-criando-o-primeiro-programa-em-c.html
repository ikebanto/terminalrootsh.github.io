Certifique-se de que o GCC tá instalado para compilá-lo, caso não esteja, basta instalar via apt-get:

{% highlight bash %}

apt-get install gcc{% endhighlight %}

Escreva o código abaixo no seu editor preferido, salve como {% highlight bash %}arquivo.c{% endhighlight %} e compile-o com o seguinte comando

{% highlight bash %}gcc arquivo.c -o arquivo{% endhighlight %}

Depois é só executar com o comando

{% highlight bash %}./arquivo{% endhighlight %}

Código

{% highlight bash %}

#include <stdio.h>     

int main() 

{     

   printf("Terminal Root\n");    

   return 0;

}  {% endhighlight %}
