Gambas é um Interpretador BASIC com extensões de orientação a objetos. Isto torna Gambas uma boa escolha para desenvolvedores VB para aqueles que querem usar os seus conhecimentos em uma base GNU. Desenvolvido em Paris por Benoît Minisini desde 1999, Gambas é liberado sob a Licença Pública Geral GNU. Várias distribuições Linux incluem Gambas em seus pacotes. "Gambas" é um acrônimo recursivo para "Gambas Almost Means BASIC" (Gambas Quase Significa BASIC). "Gamba", em língua espanhola, é o nome dado a uma espécie de camarão e por isso este é o mascote da IDE. Com Gambas é possível facilmente: Usar bancos de dados como MySQL ou PostgreSQL. Construir aplicações KDE com DCOP. Traduzir programas Visual Basic para Gambas e corrê-los sob Linux.
