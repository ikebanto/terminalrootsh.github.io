Instalar:
{% highlight bash %}apt-get install tightvncserver tightvnc-java xtightvncviewer{% endhighlight %}

Configura a senha:
{% highlight bash %}vncpasswd{% endhighlight %}

Iniciar o VNC no servidor:
{% highlight bash %}vncserver{% endhighlight %}

Caso preciso, usar o nmap:
{% highlight bash %}nmap [IP]{% endhighlight %}

Iniciar sessão:
{% highlight bash %}vncviewer [IP]:[porta]{% endhighlight %}
